# LibraryProject

This is the main Django project for learning and experimenting with Django. It was created using `django-admin startproject LibraryProject`.

## How to Run

1. Make sure you have Python and Django installed.
2. Start the development server:
   ```bash
   /home/munga/Desktop/Alx/Alx_DjangoLearnLab/.venv/bin/python manage.py runserver
   ```
3. Open your browser and go to [http://127.0.0.1:8000/](http://127.0.0.1:8000/) to see the Django welcome page.

## Project Structure

- `manage.py`: Command-line utility for Django project management.
- `LibraryProject/settings.py`: Main configuration for the Django project.
- `LibraryProject/urls.py`: URL declarations for the project.

Refer to the official Django documentation for more details: https://docs.djangoproject.com/en/stable/
